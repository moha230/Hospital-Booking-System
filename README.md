# Doctors management system

 A small doctor management system that has admin users managers. Using the new framework.

# Back end stack used 
* Hono 
* Deno 
* eta 
* zod 


## commands needed to run the server 

* deno run --watch --allow-net --allow-read --unstable-kv  app.js
